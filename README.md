# Mastersproject
Machine learning classifiers for different biases in biological miRNA datasets.
* Python 3.6
* R 3.5.2


## Requirements Python
Note: Anaconda is recommended for Windows users.
* Numpy
* Pandas
* Scikit-learn
* Seaborn
* Graphviz
* gseapy
* tqdm
* R
* rpy2

## Requirements R
* limma
* edgeR
* statmod
